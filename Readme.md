# **Telegram Bot with AI**
A Telegram bot powered by AI.

Open source for those who want such a project.

To set this up, you need to install Pip:

## For Windows:

Open your browser, search for "Download Pip", and download the executable (exe) file from the first website.

## For Linux (3 methods):
### Arch:
```bash
sudo pacman -S pip
```
---
### Fedora:
```bash
sudo dnf install pip
```
---
### Debian ( ubuntu ):
```bash
sudo apt install pip
```

## Then, in the terminal, run these commands:

```pip
pip install ollama
```
```pip
pip install aiogram
```
